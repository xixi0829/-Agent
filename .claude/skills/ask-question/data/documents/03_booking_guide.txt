商旅预订指南

一、机票预订

1. 预订时间建议
   - 国内机票：建议提前7-14天预订，价格通常最优惠
   - 国际机票：建议提前21-30天预订
   - 热门航线（如北京-上海）：建议提前3-5天即可
   - 旺季出行（如春节、国庆）：建议提前1-2个月预订

2. 预订渠道
   - 统一通过阿里商旅平台预订
   - 平台优势：企业协议价、自动审批、统一管理
   - 24小时客服支持：400-800-8888

3. 选择航班技巧
   - 对比不同时段航班价格，选择性价比高的航班
   - 避开高峰时段（如周一早上、周五晚上）可以节省费用
   - 考虑航班时间与行程安排的匹配度
   - 留出足够的转机或中转时间（建议至少2小时）

4. 改签与退票
   - 改签政策：按照航空公司规定执行
   - 特价票限制：很多特价票不能改签或改签费用较高
   - 退票费用：根据票价折扣和退票时间决定
   - 建议：购票时注意查看退改签政策，必要时选择可退改的票

5. 选座服务
   - 免费选座：大部分航空公司提供免费在线选座
   - 付费选座：如需特定座位（如紧急出口、前排）可能需付费
   - 费用承担：免费选座公司报销，付费选座需自付
   - 特殊需求：如需要轮椅、特殊餐食，可在预订时备注

6. 值机建议
   - 在线值机：航班起飞前24小时开放，建议尽早值机
   - 打印登机牌：可在线打印或机场自助打印
   - 行李托运：需要托运行李的需提前到机场办理
   - 到达机场时间：国内航班建议提前1.5小时，国际航班提前3小时

二、火车票预订

1. 预订时间
   - 开售时间：一般提前15天（根据铁路部门规定可能调整）
   - 建议：开售后尽早预订，热门线路很快售罄
   - 节假日：春运、黄金周等需提前抢票

2. 座位等级
   - 允许预订：二等座及以下
   - 特殊情况：长途夜间出行（如超过6小时）可申请卧铺
   - 商务座：需特别申请并说明理由

3. 购票渠道
   - 阿里商旅平台：优先使用，方便统一管理
   - 12306官网：也可使用，但需注意发票和报销问题
   - 代售点：紧急情况下可使用

4. 改签规则
   - 免费改签：开车前可以免费改签一次
   - 改签限制：只能改签到同一日期的其他车次，不能改日期（需退票重买）
   - 已取票的：需到车站窗口办理改签

5. 退票规则
   - 退票费：根据退票时间收取不同比例的退票费
   - 开车前15天以上：不收退票费
   - 开车前48小时至15天：收取5%退票费
   - 开车前24-48小时：收取10%退票费
   - 开车前24小时内：收取20%退票费

6. 注意事项
   - 节假日期间需提前规划，热门线路一票难求
   - 可考虑中转：如直达票售罄，可选择中转方案
   - 携带身份证：乘车需要身份证，遗失需办理临时证明
   - 到站时间：建议提前30分钟到站，以便取票和安检

三、酒店预订

1. 预订时间
   - 常规预订：建议提前3-5天预订
   - 热门城市/旺季：建议提前7-10天预订
   - 临时出差：平台支持当天预订和入住

2. 预订渠道
   - 阿里商旅平台：统一预订渠道
   - 优势：企业协议酒店、优惠价格、统一管理、快速审批

3. 酒店选择建议
   a) 位置选择
      - 优先选择会议/办公地点附近的酒店，节省交通时间
      - 考虑地铁或公交便利性
      - 避开偏僻或治安较差的区域

   b) 酒店档次
      - 优先选择企业协议酒店，享受折扣价格（通常8-9折）
      - 推荐连锁品牌：汉庭、如家、维也纳、全季等
      - 中高端选择：万豪、希尔顿、洲际等（需符合标准）

   c) 设施要求
      - 基本设施：空调、热水、WiFi、独立卫浴
      - 商务需求：办公桌、打印服务、会议室
      - 餐饮：含早餐的酒店更方便

   d) 评价参考
      - 查看其他用户的评价和评分
      - 重点关注：卫生、安全、服务、位置
      - 评分建议：选择4分以上的酒店

4. 入住规定
   - 入住时间：一般为14:00以后
   - 提前到达：可以寄存行李，房间准备好后入住
   - 入住手续：需提供身份证原件，不可使用复印件或照片
   - 押金缴纳：一般需要缴纳押金（现金或信用卡预授权）

5. 退房规定
   - 退房时间：一般为次日12:00前
   - 延迟退房：可与酒店协商延迟退房（可能收费）
   - 退房手续：结清费用、退还押金、归还房卡

6. 发票获取
   - 离店前索取发票：发票抬头为公司全称
   - 电子发票：部分酒店支持电子发票，可在平台获取
   - 补开发票：如忘记索取，可联系酒店补开（提供订单号）

7. 取消政策
   - 免费取消：一般需要提前24小时取消预订
   - 当天取消：可能收取一晚房费作为取消费用
   - 特价房：可能不可取消或取消费用较高
   - 建议：预订时仔细阅读取消政策

四、租车预订

1. 租车条件
   - 适用情况：多地出差、偏远地区、携带大量物品
   - 需提前申请并说明理由

2. 租车渠道
   - 阿里商旅平台：提供企业租车服务
   - 合作品牌：神州租车、一嗨租车等

3. 车型选择
   - 标准车型：经济型轿车（如大众朗逸、丰田卡罗拉）
   - 特殊需求：如需SUV或商务车，需说明理由并申请

4. 租车费用
   - 包含：租车费、保险费、基本服务费
   - 不包含：油费、停车费、过路费（据实报销）

5. 取还车
   - 取车：需提供本人驾驶证和身份证
   - 验车：取车时仔细检查车辆状况，拍照留证
   - 还车：按约定时间地点还车，加满油

6. 注意事项
   - 确认驾驶证有效期
   - 购买充分保险
   - 了解交通规则，避免违章
   - 如发生事故，及时联系租车公司和公司安全部门

五、预订管理

1. 行程查询
   - 登录阿里商旅平台
   - 查看"我的行程"
   - 查询所有已预订的机票、火车票、酒店

2. 行程变更
   - 变更方式：在平台上直接操作或联系客服
   - 变更费用：根据各供应商政策收取
   - 审批流程：重大变更需重新提交审批

3. 行程提醒
   - 短信提醒：出发前收到短信提醒
   - 邮件提醒：行程单发送至企业邮箱
   - APP推送：阿里商旅APP实时推送行程信息

4. 客服支持
   - 24小时客服热线：400-800-8888
   - 在线客服：平台提供在线聊天支持
   - 紧急情况：随时联系客服获得帮助

六、预订技巧与建议

1. 价格对比
   - 多对比几个时段的价格
   - 关注平台促销活动
   - 选择性价比高的产品

2. 灵活出行
   - 时间灵活的可选择便宜时段
   - 避开高峰期可节省费用
   - 考虑中转方案（如机票贵可选高铁）

3. 提前规划
   - 确定行程后尽早预订
   - 提前预订通常价格更优惠
   - 避免临时预订的高价和无房/无票情况

4. 了解政策
   - 预订前了解退改签政策
   - 特价产品注意限制条件
   - 保留预订确认信息

5. 企业协议
   - 优先选择协议酒店和协议租车
   - 享受企业折扣价格
   - 服务更有保障

七、常见问题

1. 为什么要通过阿里商旅平台预订？
   - 企业协议价格更优惠
   - 统一管理，方便审批和报销
   - 数据自动同步，减少手工录入
   - 享受24小时客服支持

2. 可以自己预订后报销吗？
   - 原则上应通过平台预订
   - 特殊情况下自行预订需提前申请并说明理由
   - 自行预订的价格超过平台价格时，按平台价格报销

3. 预订出错怎么办？
   - 立即联系客服进行处理
   - 如已产生费用，根据情况确定责任方
   - 平台错误导致的损失由公司承担

4. 如何获取企业协议价？
   - 使用企业邮箱注册和登录平台
   - 系统自动识别并提供协议价
   - 预订时注意查看"协议酒店"标识
