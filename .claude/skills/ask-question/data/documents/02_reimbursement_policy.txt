差旅费用报销规定

一、报销时间要求

1. 常规报销时限
   - 出差结束后，应在30个自然日内提交报销申请
   - 建议在出差结束后一周内提交，避免遗忘或遗失票据
   - 超过30天未报销的，需要提供书面说明并经主管审批

2. 特殊情况时限
   - 跨年度出差：应在当年12月31日前完成报销
   - 12月份出差：可在次年1月15日前报销
   - 长期驻外：可按月或按季度报销

3. 逾期处理
   - 超过30天未报销且未说明：不予受理
   - 特殊原因逾期（如生病、长期出差）：提供证明材料后可受理
   - 发票过期失效：不予报销

二、报销材料要求

1. 必需材料清单
   a) 差旅申请审批单（系统打印）
   b) 交通费用凭证
      - 火车票：车票原件或电子票打印件
      - 机票：行程单原件（必须是航空公司或代理出具的正式行程单）
      - 打车费用：发票或电子发票
      - 其他交通：租车合同、发票等

   c) 住宿费用发票
      - 酒店发票原件或电子发票
      - 发票上应显示住宿日期、天数、金额
      - 必须是增值税发票（普通发票或专用发票）

   d) 餐饮费用发票
      - 正规餐饮发票
      - 需注明消费日期和金额

   e) 其他费用发票
      - 如签证费、保险费、停车费等
      - 所有发票必须真实合法

2. 发票基本要求
   - 抬头：必须为公司全称"阿里巴巴（中国）有限公司"或所属分公司全称
   - 税号：必须填写公司统一社会信用代码
   - 内容：必须与实际消费相符，不得涂改
   - 真伪：可通过国家税务总局网站或地方税务局网站查验
   - 时效：发票必须在有效期内，一般为开票后一年内

3. 电子发票要求
   - 电子发票与纸质发票具有同等法律效力
   - 打印要求：使用A4纸打印，不得缩放，保证清晰可辨
   - 不得重复报销：系统会自动识别电子发票号码，防止重复
   - 保存备份：建议保存电子发票PDF文件，以备查验

4. 特殊票据处理
   - 火车票遗失：可持身份证到车站打印"报销凭证"
   - 机票行程单遗失：联系航空公司或代理补打（可能收费）
   - 发票遗失：需提供遗失说明和其他辅助证明（如银行流水、订单截图等）

三、报销流程

1. 在线提交
   - 登录阿里商旅系统
   - 选择"费用报销"模块
   - 关联对应的差旅申请单
   - 录入费用明细（系统会自动读取部分信息）
   - 上传发票影像（扫描或拍照，需清晰完整）

2. 票据提交
   - 在线提交后，将纸质发票原件整理好
   - 按照类别分类粘贴在报销单上（交通、住宿、餐饮等）
   - 提交到财务部指定地点（或邮寄到财务部）

3. 审批流程
   - 主管审批（1个工作日内）
   - 财务初审（2个工作日内）：审核发票真实性、合规性
   - 财务复审（1个工作日内）：审核金额准确性
   - 总监审批（超过一定金额需要）

4. 费用支付
   - 审批通过后，财务部在3-5个工作日内完成付款
   - 款项直接转入员工工资卡账户
   - 到账后会收到短信或邮件通知

四、不予报销的情况

1. 明确不予报销项目
   - 超出差旅标准的部分（未经审批）
   - 无正式发票的费用（白条、收据等）
   - 非公务产生的个人消费（如个人购物、娱乐等）
   - 未按规定审批的差旅费用
   - 逾期未报销且未说明原因的费用
   - 发票抬头错误或无抬头
   - 发票内容与实际不符
   - 虚假发票或重复报销

2. 个人消费清单
   以下费用明确为个人消费，不予报销：
   - 酒店内个人消费：如minibar、付费电影、洗衣、SPA等
   - 个人购物
   - 娱乐活动：如KTV、酒吧、景区门票等
   - 罚款：如交通违章罚款
   - 机场贵宾厅（除非特殊情况经审批）
   - 个人原因产生的改签费、退票费

3. 部分报销情况
   - 交通费超标：按标准报销，超出部分自付
   - 住宿费超标：按标准报销，超出部分自付
   - 特殊情况：需要提供详细说明和审批文件

五、报销金额计算

1. 交通费计算
   - 火车票：按票面金额报销
   - 飞机票：按行程单金额报销（含税费、燃油费）
   - 打车费：按发票金额报销
   - 租车费：按合同金额和实际天数计算

2. 住宿费计算
   - 按实际住宿天数计算
   - 住宿费包含：房费、早餐（如含）、服务费、税费
   - 不包含：个人消费、额外加床费用等

3. 餐饮费计算
   - 按实际发票金额报销
   - 每餐上限100元
   - 业务招待：按实际金额报销，需提供业务说明

4. 补贴计算
   - 国内出差：无补贴
   - 国际出差：按天计算，不同国家标准不同

六、特殊费用报销

1. 行李费用
   - 托运行李费：航空公司免费额度内不产生费用
   - 超重行李费：因工作需要（如携带设备、样品）可报销，需说明
   - 行李丢失赔偿：按航空公司规定处理，公司可协助

2. 通讯费用
   - 国内通话：不单独报销（计入个人话费补贴）
   - 国际漫游：可报销业务相关通话费用，需提供通话记录
   - 国际漫游数据流量：开通套餐后可报销

3. 签证费用
   - 签证费：全额报销
   - 签证照片费：可报销
   - 签证加急费：紧急出差需要加急的可报销
   - 签证服务费：选择经济的代办服务，可报销

4. 保险费用
   - 公司统一购买的差旅意外险：无需个人报销
   - 额外购买的保险：经审批可报销

5. 其他合理费用
   - 会议资料打印费：可报销
   - 快递费用：寄送业务相关物品可报销
   - 停车费：业务拜访产生的停车费可报销
   - 过路过桥费：租车或自驾出差产生的可报销

七、发票管理

1. 发票查验
   - 财务部会对所有发票进行真伪查验
   - 查验渠道：国家税务总局全国增值税发票查验平台
   - 假发票：一经发现，不予报销，并按公司规定处理当事人

2. 发票保存
   - 纸质发票：由财务部统一保存，保存期限7年
   - 电子发票：系统自动保存，保存期限长期
   - 员工应自行保存发票副本或照片，以备查询

3. 发票丢失处理
   - 遗失发票：需出具书面说明，经主管签字确认
   - 需提供辅助证明：如订单截图、支付记录、银行流水等
   - 金额较大的发票遗失：需提供更详细的证明材料

八、常见问题处理

1. 发票抬头错误
   - 可联系开票方重新开具（需要原发票作废或红冲）
   - 无法重开的：不予报销
   - 预防措施：开票前仔细核对公司全称和税号

2. 发票金额超标
   - 未经审批超标：按标准金额报销
   - 经审批超标：按实际金额报销
   - 超标部分需个人承担的：在报销时自动扣除

3. 多人共同出差
   - 发票可由一人代为统一开具
   - 报销时需分摊到各人账户
   - 需提供同行人员名单和分摊明细

4. 报销金额错误
   - 提交前仔细核对
   - 提交后发现错误：可撤回修改（未审批状态下）
   - 已审批的：需联系财务部处理

九、审计与监督

1. 定期审计
   - 财务部定期对差旅费用进行审计
   - 重点审查：发票真实性、费用合规性、审批流程完整性

2. 随机抽查
   - 对报销金额较大的或频繁出差的员工进行抽查
   - 抽查内容：发票、行程、审批流程

3. 违规处理
   - 发现违规行为：按公司规定处理
   - 严重违规：可能影响个人信用记录
