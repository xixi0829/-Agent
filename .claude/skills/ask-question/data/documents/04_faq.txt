商旅常见问题解答（FAQ）

一、出差申请相关

Q1: 出差需要提前多久申请？
A1: 一般出差需要提前至少3个工作日提交申请。紧急出差可以特事特办，但需要在申请中详细说明紧急原因，并在返回后3日内补齐审批手续。国际出差建议提前1-2周申请，以便办理签证等手续。

Q2: 出差申请被驳回了怎么办？
A2:
- 首先查看驳回原因，通常会有具体说明
- 根据驳回原因修改申请内容
- 可与主管沟通了解具体问题
- 修改后重新提交申请
- 如有争议，可向人力资源部咨询

Q3: 临时需要延长出差时间怎么办？
A3:
- 在阿里商旅系统中提交行程变更申请
- 说明延长原因和延长天数
- 经主管审批通过后方可延长
- 延长产生的费用按原标准报销
- 建议提前提交变更申请，避免影响后续行程

Q4: 可以提前一天出发或晚一天返回吗？
A4: 可以，但有以下规定：
- 如果是个人原因（如想顺便旅游）：额外产生的住宿费用需自理
- 如果是公务需要（如航班时间限制、业务安排等）：需在申请中说明理由，经审批后可报销
- 提前出发或延后返回的日期算作出差时间还是个人时间，需在申请中明确标注
- 建议在申请时就注明提前/延后计划，避免后期争议

二、交通出行相关

Q5: 出差可以开私家车吗？
A5: 可以，但需要注意：
- 需在申请中注明使用私家车及原因（如携带大量物品、多地出差等）
- 油费、过路费、停车费可按实际发生报销（需提供发票）
- 不报销车辆折旧、保养、保险等费用
- 如发生事故，个人商业保险优先理赔，不足部分公司可协助
- 建议购买充足的商业保险

Q6: 机票可以选择头等舱或商务舱吗？
A6: 原则上不可以，但以下特殊情况可以申请：
- 身高超过1.85米，经济舱座位确实不适
- 孕妇（需提供怀孕证明）
- 健康原因（需提供医生证明）
- 紧急公务需要在飞行途中处理重要工作
- 需要提前申请并获得审批，未经审批自行升舱的差额部分不予报销

Q7: 打车费用有限制吗？
A7:
- 没有单次打车金额限制，但需合理
- 允许打车的场景：机场/火车站接送、业务拜访、携带重要物品等
- 市内办公鼓励使用公共交通或公司班车
- 频繁打车或金额异常高的会被重点审核
- 建议优先使用企业合作的打车软件，享受协议价

Q8: 航班延误或取消了怎么办？
A8:
- 第一时间联系航空公司了解情况和解决方案
- 如需改签或退票，通过阿里商旅平台操作
- 如需临时住宿，选择机场附近经济型酒店
- 保留所有凭证：延误证明、改签凭证、住宿发票等
- 因延误产生的合理费用（如改签费、住宿费）可以报销
- 及时通知主管和相关业务方，调整工作安排

三、住宿相关

Q9: 酒店价格超过标准怎么办？
A9:
- 如果是旺季或特殊时期导致价格普遍上涨：可在申请中说明情况，经审批后可适当上浮
- 如果是个人选择了更高档酒店：按标准报销，超出部分自理
- 建议优先选择企业协议酒店，价格通常在标准范围内
- 如实在找不到符合标准的酒店，可提前申请预算调整

Q10: 到店后发现房间有问题怎么办？
A10:
- 卫生问题：及时要求换房或换酒店
- 设施故障：要求酒店及时维修或换房
- 安全隐患：立即换房或换酒店，并向酒店提出投诉
- 与预订不符：核对订单信息，要求酒店按订单提供
- 无法解决的问题：联系阿里商旅客服协助处理
- 如需换酒店产生额外费用：保留证据，公司可视情况报销

Q11: 可以和同事拼房吗？
A11:
- 不建议拼房，公司为每位出差员工提供独立住宿
- 特殊情况下（如房源紧张）可以拼房，但必须同性
- 拼房不影响各自报销，每人按实际住宿天数报销
- 发票可由一人代开，报销时注明拼房情况

四、费用报销相关

Q12: 出差期间的所有餐费都能报销吗？
A12:
- 早餐：如包含在酒店费用中，不单独报销；如不含，可报销不超过30元/餐
- 午餐和晚餐：可报销，每餐不超过100元
- 业务招待餐：按实际发生报销，需提供业务说明和参与人员名单
- 个人零食、饮料等：不予报销
- 酒水：非业务招待不予报销

Q13: 出差期间生病产生的医疗费用可以报销吗？
A13:
- 医疗费用不属于差旅费用报销范围
- 应通过医疗保险或商业保险报销
- 公司为出差员工购买了差旅意外险，可咨询保险理赔
- 因出差导致的工伤，按工伤保险相关规定处理
- 建议出差前准备常用药品，注意身体健康

Q14: 出差途中产生的行李托运费可以报销吗？
A14: 可以，但需要注意：
- 航空公司免费额度内的托运：不产生费用，无需报销
- 超重行李费：如果是因工作需要携带设备、样品等导致超重，可以报销，需在报销时说明
- 个人物品超重：不予报销
- 需要提供托运费发票或收据

Q15: 遗失发票怎么办？
A15:
- 尽快联系开票方补开（可能需要支付手续费）
- 如无法补开，需提供：
  - 书面遗失说明（注明原因、时间、金额等）
  - 主管签字确认
  - 辅助证明材料（如订单截图、支付记录、银行流水等）
- 财务部会根据情况审核决定是否接受
- 金额较大的（如超过500元）需要更详细的证明材料
- 建议养成及时拍照备份发票的习惯

五、个性化需求相关

Q16: 出差期间可以携带家属吗？
A16:
- 不可以。公司差旅政策仅报销员工本人的费用
- 家属随行产生的所有费用（交通、住宿、餐饮等）需个人承担
- 如家属随行，预订时需注明，避免影响差旅报销
- 特殊情况（如长期驻外需要家属陪同）需向公司申请，获得批准后另行处理

Q17: 我有特殊饮食要求（清真、素食等），怎么办？
A17:
- 餐饮报销标准同普通员工
- 可根据自己的饮食习惯选择合适的餐厅
- 如因特殊饮食导致费用略高，可在合理范围内报销
- 飞机上的特殊餐食可在订票时备注（如清真餐、素食餐等）

Q18: 我想升级酒店或机票，差额自己出可以吗？
A18: 可以，但需要注意：
- 需要在预订时明确标注自费升级
- 报销时只能报销标准范围内的费用
- 差额部分需要单独支付（不能在报销中扣除）
- 发票需要分开开具或在报销时注明自费部分

六、国际出差相关

Q19: 国际出差需要办理哪些手续？
A19:
- 护照：确保有效期在6个月以上
- 签证：根据目的地国家要求办理（公司可协助）
- 疫苗：部分国家要求接种疫苗或提供健康证明
- 保险：公司会统一购买国际差旅保险
- 外汇：可在银行兑换或使用信用卡
- 通讯：开通国际漫游或购买当地电话卡

Q20: 国际出差的费用标准和国内一样吗？
A20: 不一样，主要差异：
- 住宿标准：根据目的地国家和城市有不同标准
- 餐饮标准：国际出差有每日补贴，标准因国家而异
- 通讯费用：国际漫游费可报销
- 签证费用：全额报销
- 详细标准可查看《国际差旅管理办法》或咨询人力资源部

七、其他常见问题

Q21: 周末或节假日出差有补贴吗？
A21:
- 周末或节假日出差属于加班，可以申请调休
- 调休天数等于实际出差的周末/节假日天数
- 具体调休政策请咨询人力资源部
- 差旅费用报销标准与工作日出差相同

Q22: 如何申请企业协议酒店？
A22:
- 无需单独申请，使用企业邮箱登录阿里商旅平台即可
- 系统会自动显示协议酒店，并标注"协议"标识
- 协议酒店价格通常比市场价低10%-20%
- 协议酒店服务更有保障，推荐优先选择

Q23: 可以使用个人里程或积分预订机票/酒店吗？
A23: 可以，但需要注意：
- 使用个人里程或积分预订的部分，公司不予报销
- 只报销实际支付的现金部分
- 需要提供支付凭证和发票
- 报销时需注明使用了里程或积分

Q24: 出差申请单丢失了怎么办？
A24:
- 在阿里商旅系统中可以随时查询和打印历史申请单
- 登录系统 → "我的差旅" → 选择对应申请 → 打印
- 如系统中也找不到，联系人力资源部或IT部门恢复数据

Q25: 对差旅政策有建议或意见，向谁反馈？
A25:
- 可以通过以下渠道反馈：
  - 向直属主管或人力资源部反馈
  - 通过公司内部沟通渠道（如钉钉群）提出
  - 在公司季度员工座谈会上提出
  - 发送邮件至 hr@alibaba-inc.com
- 公司会定期收集和评估员工建议，不断优化差旅政策

Q26: 出差期间可以顺便办理私人事务吗？
A26: 可以，但需要明确区分：
- 出差期间的非工作时间（如晚上、周末）可以自由安排
- 因办理私人事务产生的费用不能报销
- 如因私人事务需要延长出差时间，额外产生的费用需自理
- 要确保私人事务不影响工作任务的完成

Q27: 如何确认差旅费用是否符合标准？
A27:
- 查阅《差旅管理办法》了解详细标准
- 使用阿里商旅平台预订，系统会自动标注是否符合标准
- 预订前可咨询人力资源部或财务部
- 报销时系统会自动审核是否符合标准

Q28: 出差期间的车辆违章罚款可以报销吗？
A28:
- 不可以。交通违章罚款属于个人行为，不予报销
- 无论是租车、私家车还是出租车产生的违章，都由个人承担
- 建议出差期间严格遵守交通规则，避免违章
