### ✅ Q2: 如何报销差旅费用？需要哪些材料？

**耗时**: 27.02秒

**回答**:

```
### 差旅费用报销流程及所需材料


#### **一、报销时间要求**
1. 
**常规报销时限**：出差结束后30个自然日内提交报销申请，建议一周内提交，避免遗忘或遗失
票据。  
2. **特殊情况时限**：  
   - 跨年度出差：当年12月31日前完成报销；  
   - 12月份出差：次年1月15日前报销；  
   - 长期驻外：按月或按季度报销。  
3. **逾期处理**：  
   - 超过30天未报销且无说明：不予受理；  
   - 特殊原因（如生病、长期出差）：提供证明材料后可受理；  
   - 发票过期失效：不予报销。


#### **二、必需报销材料清单**
1. **差旅申请审批单**：系统打印（需提前申请并通过审批）。  
2. **交通费用凭证**：  
   - 火车票：车票原件或电子票打印件；  
   - 机票：航空公司或代理出具的正式行程单原件；  
   - 打车费用：发票或电子发票；  
   - 其他交通（如租车）：租车合同+发票。  
3. **住宿费用发票**：  
   - 酒店发票原件或电子发票；  
   - 需显示住宿日期、天数、金额；  
   - 必须为增值税发票（普通发票或专用发票）。  
4. **餐饮费用发票**：  
   - 正规餐饮发票；  
   - 需注明消费日期和金额。


#### **三、特殊情况处理**
1. **超标准报销**：  
   - 
确因特殊原因需超标准（如超标30%以内），须提前申请并说明理由，经主管审批；超标30%以上
需总监审批；  
   - 未经审批的超标部分不予报销。  
2. **行程变更**：  
   - 及时在系统中更新行程，延长超过3天需重新提交审批。  
3. **其他费用报销**（可选，按需提供）：  
   - **行李费**：超重行李费因工作需要携带设备等导致，需提供说明；  
   - **停车费**：业务拜访停车费可报销，需提供停车票据（机场停车费不予报销）。


**说明**：报销时需确保材料齐全、真实有效，超时或超标准情况需额外审批，未按要求处理可
能导致不予报销。
```

### ✅ Q4: 机票应该提前多久预订？有什么注意事项？

**耗时**: 25.95秒

**回答**:

```
机票预订提前时间及注意事项如下：

### 一、预订时间建议
- 国内机票：建议提前7-14天预订，价格通常最优惠；
- 国际机票：建议提前21-30天预订；
- 热门航线（如北京-上海）：建议提前3-5天即可；
- 旺季出行（如春节、国庆）：建议提前1-2个月预订。

### 二、注意事项
1. 
**预订渠道**：统一通过阿里商旅平台预订，可享受企业协议价、自动审批及统一管理服务，客
服支持400-800-8888。
2. 
**选择航班技巧**：对比不同时段航班价格，避开高峰时段（如周一早上、周五晚上），考虑航
班时间与行程安排的匹配度，留出至少2小时中转时间。
3. 
**退改签政策**：购票时需注意查看退改签政策，特价票通常不可改签或改签费用较高，建议选
择可退改的票。
4. 
**选座服务**：大部分航空公司提供免费在线选座（公司报销），如需特定座位（如紧急出口、
前排）可能需付费（需自付），特殊需求（如轮椅、特殊餐食）可在预订时备注。
5. 
**机场时间**：国内航班建议提前1.5小时到达机场，国际航班提前3小时到达，以便办理值机、
安检等手续。
```

### ✅ Q9: 航班取消了怎么办？紧急情况联系谁？

**耗时**: 26.91秒

**回答**:

```
航班取消后的处理办法：
1. 立即联系航空公司了解取消原因和退改方案；
2. 
评估对行程的影响：若为紧急公务，考虑改签最近的航班（可能需中转或更高费用）；若时间允
许，可改签其他日期或选择其他交通方式（如高铁）；
3. 通过阿里商旅平台操作退票或改签，优先选择航空公司提供的免费改签；
4. 及时通知主管和业务相关方，调整工作安排；
5. 保留所有凭证（如退改签凭证、费用票据等）用于报销；
6. 如需紧急出行且无合适航班，可申请应急审批，选择其他交通方式。

紧急情况下应联系：航空公司（了解取消原因及退改方案）、主管及业务相关方（调整工作安排
）。
```
