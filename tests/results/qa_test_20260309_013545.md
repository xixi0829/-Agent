**耗时**: 35.5秒

**回答**:

```
差旅费用报销需遵循以下流程和材料要求：

### 一、报销时间要求
1. **常规时限**：出差结束后30个自然日内提交报销申请，建议一周内提交，避免遗忘；
2. **特殊情况**：
   - 跨年度出差：当年12月31日前完成报销；
   - 12月出差：可延至次年1月15日前报销；
   - 长期驻外：按月或季度报销；
3. **逾期处理**：超30天未报销且无说明不予受理，特殊原因（如生病）需提供证明材料可受理，发票过期失效不予报销。

### 二、必需报销材料
1. **差旅申请审批单**：系统打印；
2. **交通费用凭证**：
   - 火车票：原件或电子票打印件；
   - 机票：航空公司/代理出具的正式行程单原件；
   - 打车费：发票或电子发票；

3. **住宿费用发票**：酒店开具的原件或电子发票，需显示住宿日期、天数、金额，且为增值税发票（普通或专用）；
4. **餐饮费用发票**：正规发票，需注明消费日期和金额。

### 三、特殊费用说明（需额外材料）
1. **停车费**：业务拜访停车费可报销，需提供停车票据（机场停车费不予报销）；
2. **超重行李费**：因工作携带设备等导致超重，需在报销时说明理由。

### 四、其他注意事项
- 超标准消费需提前申请并说明理由，30%以内由主管审批，30%以上需总监审批；
- 行程变更（如延长超3天）需重新提交审批；
- 发票过期或遗失需补开或提供书面说明及证明材料（如订单截图、支付记录）。
```

### ✅ Q4: 机票应该提前多久预订？有什么注意事项？

**耗时**: 23.07秒

**回答**:

```
### 机票预订提前时间建议：
- **国内机票**：建议提前7-14天预订，价格通常最优惠；
- **国际机票**：建议提前21-30天预订；
- **热门航线（如北京-上海）**：建议提前3-5天即可；
- **旺季出行（如春节、国庆）**：建议提前1-2个月预订。

### 注意事项：
1. **预订渠道**：统一通过阿里商旅平台预订，享受企业协议价、自动审批及统一管理服务，客服支持电话400-800-8888。
2. **选择航班技巧**：对比不同时段航班价格，避开高峰时段（如周一早上、周五晚上），考虑航班时间与行程匹配度，中转航班预留至少2小时转机时间。
3. **退改签政策**：按照航空公司规定执行，特价票通常限制改签或改签费用较高，购票时建议查看退改签政策。
4. **选座服务**：大部分航空公司提供免费在线选座，紧急出口、前排等特殊座位可能需付费，免费选座可由公司报销，付费需自付。
5. **值机与到达机场**：
   - 在线值机：航班起飞前24小时开放，建议尽早值机；
   - 登机牌：可在线打印或机场自助打印；
   - 行李托运：需提前到机场办理；
   - 到达机场时间：国内航班建议提前1.5小时，国际航班提前3小时。
```
